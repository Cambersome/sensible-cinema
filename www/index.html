Welcome to the (work in progress) home page of Sensible Cinema, the DVD scene deleter.

More detail about the program can be found in the README.


Download

Please download the latest version it from the releases directory,
the latest version can be found here.
Unzip it (right click -> Extract all).  
Now open the newly created "sensible-cinema-x.x.x" folder,
and run the contained file "run sensible cinema.bat".
You will be initially prompted to accept the license agreement and also install a few (non included) dependency programs.
Now insert a DVD you have, and choose from sensible cinema's available options.
Enjoy!

Sensible cinema is currently for windows only, ping me if you want this changed.

If you want to create your own "delete list" for a DVD that doesn't have one yet, 
then run the contained file "edit or create sensible cinema delete list files.bat"
See also the how to create file for instructions on creating a delete list.


Creating your own edited DVD

If you so desire, you could create an edited DVD from the edited file that sensible cinema gives you.
See how to create a DVD from a video file in windows [full size] for a demo of how to use the free DVD flick to burn a video file to disk.  See also LICENSE.TXT

Windows DVD Maker might help, too.
