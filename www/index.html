Welcome to the (work in progress) home page of Sensible Cinema, the video scene skipper/muter.

A description of the program can be found here.


Download

Please download (save) the latest version via latest-sensible-cinema.zip.
NB that you want to download from here (latest-sensible-cinema.zip), not from github (source only).
After downloading it, unzip it (show in explorer -> right click on the .zip file -> Extract all -> Extract).  
Now open the newly extracted "sensible-cinema-x.x.x" folder,
and run the file "run sensible cinema".
You will be initially prompted to accept the license agreement(s) and also install a few (non included, separate project) dependencies.
Now insert a DVD you own, and then choose from sensible cinema's available options (buttons).

You can see the list of the DVD delete lists available here (all of them created earlier by volunteers like you).  
If yours is there, then you should be good to go.  Enjoy!

If you want to create your own "delete list" for a DVD that doesn't have one yet, 
see the how to create file for instructions on creating a new delete list. 
Then run the contained file "edit or create sensible cinema delete list files.bat" to create and edit your file.  Don't forget to
upload it back to us when you're done, so others can benefit from your work.
